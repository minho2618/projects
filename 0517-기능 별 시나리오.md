- 방 조회 기능
    1. 모든 방 정보 확인하기
    - 현재 게스트하우스의 모든 방 정보를 확인한다.
    - 해당 방의 예약 가능 여부는 확인하지 않는다.
    2. 인기있는 방을 확인하기
    - 오늘을 기준으로 약 1년 간 게스트하우스에서 가장 예약이 많은 방을 확인한다.
    - 예약이 가장 많이된 상위 3위만 확인한다.
    3. 예약이 마감된 방 확인하기
    - 오늘을 기준으로 방마다 예약이 불가능한 날짜를 확인한다. 
    - 조회는 오늘을 기준으로 1달
    4. 원하는 날짜의 남아있는 방 확인하기
    - 원하는 날짜를 입력하면 남아있는 방들을 확인한다.
    - 방의 이름만 조회한다.
    5. 원하는 날짜의 남아있는 방 중 인원수에 맞는 방 확인하기
    - 원하는 날짜를 입력하면 남아있는 방들을 확인한다.
    - 이때 인원수를 수용가능한 방만 조회한다.
- 조식 메뉴 조회 기능
    6. 오늘의 조식 메뉴 확인하기
    - 조식메뉴를 조회한다.
- 방 예약 기능
    7. 예약하기
    - 먼저 예약할 년도, 월, 일을 입력한다.
    - 그런 뒤, 예약자의 이름, 성별, 전화번호를 입력한다.
    - 예약자의 성별에 따른 예약가능한 방 리스트를 조회한다.
    - 방 이름을 입력해 예약할 방을 선택한다.
    - 조식을 먹을 지를 선택한다.
    - 예약 완료가 된다면 예약 번호를 받는다.
- 예약 정보 수정
